# node-jwt-login
A simple JWT login example in Node JS.

<p>This approach is to show a SPA login using JWT to redirect to a private dashboard area.</p>

## Install
```
git clone https://github.com/nuntius-rex/node-jwt-login.git
```

## Run
```
npm start
```
